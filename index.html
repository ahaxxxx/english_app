这版已经支持 Supabase 同步，但前提是你已经按说明配置好项目、表结构和邮箱登录。

如果暂时没配置 Supabase，页面仍然会退回本地模式，导出导入功能也能继续用。

适合 GitHub Pages 部署。手机端建议“添加到主屏幕”，桌面端可直接安装为独立应用。
